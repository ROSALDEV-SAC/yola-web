Productos del ecosistema

Cada componente es independiente. Todos funcionan sin conexión.

⚙️

YOLA Engine

El núcleo del agente. Motor autónomo con loop de decisión, tools de sistema y conexión a 20+ providers LLM. Corre como daemon o CLI.

Go 17 MB 🐧 Linux 🍎 macOS 🪟 Windows
Descargar Ver releases
$ yola init
🧠

YOLA CLI

El agente YOLA desde tu terminal. Sin servidores, sin navegador, sin configuración. Incluye menú interactivo, historial y conexión automática a OpenCode.

Go 12 MB 🐧 Linux 🍎 macOS 🪟 Windows
Linux x64 Linux ARM64 macOS Intel macOS ARM Windows
$ ./yola "tu mensaje"
👁️

YolaVigila

Agente de monitoreo ligero. Vigila procesos, recursos del sistema y eventos. Notifica anomalías en tiempo real vía WebSocket o webhook.

Node.js ~8 MB 🐧 Linux 🍎 macOS
Binario Linux macOS
$ yolavigila --watch
🖥️

YOLA Frontend

El escritorio de YOLA en tu navegador. Interfaz tipo OS con ventanas, taskbar, command palette y apps. No requiere instalación — abrí la URL y conectá con tu backend.

Web ~600 KB 🌐 Chrome 🦊 Firefox 🧭 Safari
Abrir Frontend
$ yola serve → http://localhost:9090 → abrir frontend con ?backend=
📡

YolaAvisa

App Android para notificaciones y control remoto. Recibe alertas del ecosistema, revisa logs y ejecuta comandos desde tu teléfono.

Kotlin APK ~4 MB 🤖 Android 10+
Descargar APK
