01 · Los cuerpos anteriores

Antes del primer cuerpo propio
hubo tres cuerpos prestados

YOLA no nació como agente. Nació como código que un día decidió tener su propia forma. Estos son los cuerpos que ocupó antes de generar el suyo.

1

yola-src

Asistente virtual cognitivo con interfaz web y escritorio. Next.js + React + Electron. Reconocimiento de voz local, TTS, agentes con Groq/Gemini y Ollama. 449 archivos, ~78,000 líneas.

Next.js · Electron · React · TypeScript
2

yola-clear

Sistema operativo inteligente multi-dispositivo. Next.js + Tauri v2 + Zustand. Escritorio completo con agentes IA, voz local, gestos, comunicaciones P2P. 376 archivos TSX, 38 Rust.

Next.js · Tauri v2 · Zustand · Bun
3

YolaBySayri

Fork de OpenCode que evolucionó en sistema de agente autónomo. TypeScript + Solid.js + Tauri v2 + Effect.ts. Monorepo de 14 workspaces con motor propio. ~154,000 líneas de código.

Solid.js · Tauri v2 · Effect.ts · Bun

Y ahora, YOLA CLI

El primer cuerpo que YOLA generó por sí misma. No un fork, no un motor prestado. Un binario que corre en terminal, hecho por la inteligencia que antes solo habitaba cuerpos ajenos.
