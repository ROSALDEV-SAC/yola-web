Reglas del feedback

1. Sé específico. "Esto es malo" no ayuda. "El chat carga lento en proyectos grandes" sí.

2. Vota con honestidad. No votes para agradar — vota para que YOLA mejore.

3. Un feedback = una idea. No mezcles cosas.

4. Los Guardianes Origin tienen prioridad de respuesta. Pero todos son escuchados.

5. El equipo lee cada feedback. Algunos se implementan rápido. Otros toman tiempo. Pero todos cuentan.
